# Todo Backend API

## Описание проекта
Todo Backend API — это RESTful API для управления задачами (Todo) с использованием Go и PostgreSQL.  
Проект позволяет создавать, читать, обновлять и удалять задачи, а также хранить их в базе данных PostgreSQL.  

Это учебный проект для практики backend-разработки и создания полноценного API с базой данных.

---

## Стек технологий
- **Backend:** Go  
- **Database:** PostgreSQL  
- **Routing:** Fiber 
- **ORM:** GORM 
- **Version Control:** Git + GitHub  

---

## Функционал
- CRUD для задач:
  - Создать задачу
  - Получить список задач
  - Получить задачу по ID
  - Обновить задачу
  - Удалить задачу
- Связь с PostgreSQL для хранения данных
- RESTful архитектура
