# Todo Backend API

## Project Description
Todo Backend API is a RESTful API for managing tasks (Todo) built with **Go** and **PostgreSQL**.  
The project allows creating, reading, updating, and deleting tasks, with data stored in a PostgreSQL database.  

This is an educational project for practicing backend development and building a full-featured API with a database.

---

## Tech Stack
- **Backend:** Go  
- **Database:** PostgreSQL  
- **Routing:** Fiber  
- **ORM:** GORM  
- **Version Control:** Git + GitHub  

---

## Features
- CRUD operations for tasks:
  - Create a task
  - Get all tasks
  - Get a task by ID
  - Update a task
  - Delete a task
- Integration with PostgreSQL for data storage
- RESTful architecture
